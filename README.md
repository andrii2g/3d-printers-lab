# 3d-printers-lab

3D printers documentation, settings, etc

This repository is intended to keep printer knowledge in one place while still separating machine-specific information. Each printer has its own folder with isolated notes, configs, slicer profiles, maintenance history, and reference links so changes on one machine do not get mixed into another.

## What belongs here

- Slicer profiles
- Printer configs
- Calibration notes
- Maintenance logs
- Mods and upgrades
- Manuals and useful links

## Current printers

- Voron 0.2
- Ender 3 V2
- Sovol SV08

## Repository layout

- `docs/` for cross-printer knowledge and shared procedures
- `printers/` for machine-specific records, configs, and notes
