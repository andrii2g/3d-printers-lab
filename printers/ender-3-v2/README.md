# Ender 3 V2

## Printer overview

Machine-specific workspace for the Ender 3 V2. Use this folder to keep setup notes, calibration history, profiles, and maintenance records separate from other printers.

## Hardware / main components

## Slicer setup

- Store machine-specific slicer profiles in `slicer-profiles/`
- Record profile naming rules and slicer-specific notes here
- Track changes that affect quality, speed, or reliability

## Configuration files

- Store firmware-related configs, printer setting exports, and helper notes in `configs/`
- Keep a note of which files are current and which are retired
- Document significant changes that should be reproducible later

## Calibration notes

- Put detailed reports and tuning logs in `calibration/`
- Track bed leveling approach, flow tuning, temperature tuning, and dimensional checks
- Link calibration outcomes to slicer or config changes where applicable

## Manuals / references

- Store official docs, community references, and vendor links in `manuals/`
- Add brief context so important references are easy to find later
