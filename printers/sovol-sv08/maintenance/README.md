# Maintenance

Use this folder for dated maintenance notes, service procedures, repairs, inspections, and software-maintenance follow-up tasks.

Structure:

- `routine/` for scheduled maintenance and normal service work
- `incidents/` for failures, symptoms, fixes, and follow-up actions
- `assets/` for screenshots, photos, and videos linked from maintenance entries
- `parts-history.md` for a quick running list of replaced or installed parts
