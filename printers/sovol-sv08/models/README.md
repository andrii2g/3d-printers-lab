# Models

Use this folder for a small set of printer-specific models that are worth keeping with the SV08 documentation.

Keep this area limited to:

- calibration and test models
- utility parts used during maintenance or setup
- tracked project builds such as enclosure parts

Do not use this folder for a large STL library. If the collection grows, move the bulk of models to a separate models repository.

## Structure

- `calibration/` for first-layer, flow, temperature, retraction, dimensional, and resonance test models
- `projects/` for tracked build projects such as enclosure parts
