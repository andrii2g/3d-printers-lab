# Calibration Models

Use this folder for small test models used to tune the printer.

Recommended subfolders:

- `first-layer/`
- `flow/`
- `temperature/`
- `retraction/`
- `dimensional/`
- `resonance/`

Current known model:

- `first-layer/SV08_first_layer_square_60x60x0p20mm.stl` for checking first-layer squish, adhesion, and Z-offset consistency across a 60x60 area
